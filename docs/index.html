1. Conexión con GitHub

El token no se guarda dentro de app.py ni se publica en el JSON. Para publicar necesita permiso Contents: Read and write únicamente sobre este repositorio.
Bloqueo rápido: al pulsar “Bloquear ahora”, el panel publica automáticamente. Con revisión de 5 segundos, la PC suele cerrarse aproximadamente entre 5 y 10 segundos, dependiendo de Internet y de la propagación de GitHub.
